# linux 笔记
## ls [-a -l -h] [文件路径]
> 打印路径下所有文件名  
>> -a显示全部（包括隐藏文件）  
>> -l 竖向打印  
>> -h 文件大小单位化  
## cd [文件路径]
转换到**工作目录
## pwd 
打印当前目录路径  
## 路径
1. 默认相对路径  
2. >特殊路径符  
    >> . 表示当前目录  
    >> .. 表示退回上一级  
    >> ~ 表示home目录  
## mkdir [-p] 路径
-p 适用于自动创建多级目录  
## touch 路径
创建文件，路径带上格式  
## 区分文件和文件夹  
1. 深色为文件夹，浅色为文件
2. ls打印 d 为文件夹；- 为文件  
## cat 路径
查看文件  
## more 路径
查看文件，可翻页,空格翻页，q退出
## cp [-r] 被复制文件路径 复制到路径
-r 表示复制文件夹
## mv 被移动 移动到
移动；若“移动到”不存在，则为改名
## rm [-r -f] 被删除文件（可有多个）
-r 文件；-f强制删除（管理员运行会有确认提示）  
支持通配符，*hyf 表以hyf结尾的；*hyf*表含有hyf的
## which 查找的命令
查找命令位置
## find 起始路径 -name "文件名“
-name 表示按文件名查找，支持通配符
## find 起始路径 -size +|-n[kMG]
-size 按大小搜索 +表大于 ；-表小于；k小写
## grep [-n] “关键字” 文件名
在文件中筛选含关键字的号；-n显示行号
## wc [-c -m -l -w] 文件名
统计文件的 行数 单词数 字节数  
-c bytes 数  
-m 字符数  
-l 行数  
-w 单词数 
## 管道符 |
左边的输出作为右边的输入；可以嵌套
## echo "内容"
打印内容
## `
飘号，将内容作为命令，而不是字符串
## > >>
左侧内容写入右侧文件  
\> 覆盖写入  
\>> 追加写入
## tail [-f -num] 文件
查看文件尾部信息  
-f 持续跟踪  
-num 查看行数，默认十行
## ps [-e -f] 
查看进程；一般直接-ef; 配合|grep 使用
## kill [-9] 进程ID
结束进程，-9 强制结束
## top
还有一大堆选项和参数，显示当前主机状态
## reboot
重启 (还有一堆选项)
## shutdown
关机 (还有一堆选项)
